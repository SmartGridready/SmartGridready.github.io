Welcome to the SmartGridready Documentation
=============================================================================

SmartGridready provides a standardized framework for integrating flexible energy
devices into Energy Management Systems and power grid processes.

.. hint::

    Start with :ref:`home` for a full chapter overview, or jump directly to a
    chapter from the links below.

Quick Start
-----------

If you are new to SmartGridready, begin with these chapters:

1. :ref:`introduction` - Core concepts, architecture, and terminology.
2. :ref:`functional-profiles` - How a Functional Profile defines a selection of Data Points that together enable a specific functionality.
3. :ref:`product-description-eid` - How Product Description Files describe a Product and provide the information needed to use it through the generic SmartGridready API.
